== Description ==

This plugin provides ability to add and manage property listings at your WordPress website.

== Installation ==

1. Upload `datum-property-listing` zip file to the `/wp-content/plugins/` directory
2. Extract the plugin folder from zip file
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Copy the template file name "property-listing-template" to your active theme's folder.
5. Create a new page and assign this template "Datum Property Listing" to that page.
6. Make sure you already activated the plugin.
